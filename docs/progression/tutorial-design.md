# 新手引导与教学设计

> 配套文档：`quest-system.md`（7 日破冰计划）、`play.md`（核心玩法）、`ui-flow.md`（界面流程）
> 目标：定义新玩家从首次登录到独立游戏的完整教学流程，确保留存率

---

## 一、设计原则

| 原则 | 说明 |
|---|---|
| **边玩边学** | 不用文字说明，通过实际操作学习 |
| **即时反馈** | 每步操作后立即展示结果（如获得卡牌、伤害数字） |
| **渐进式** | 先教核心（战斗），再教养成（升星/装备），最后教社交（公会） |
| **可跳过** | 老玩家/回归玩家可跳过已学内容 |
| **不阻塞** | 教学过程中允许玩家做其他操作（如查看图鉴） |

## 三、阶段一：核心战斗教学（Day 1，前 10 分钟）

### 3.1 教学流程

```
Step 1: 登录 → 选择服务器 → 创建角色（输入昵称）
         ↓
Step 2: 剧情引入（寒冰王座，1 分钟动画）
         ↓
Step 3: 赠送初始卡牌（3 张：1 战士 + 1 法师 + 1 辅助）
         ↓
Step 4: 进入教学关（1-1 关卡）
         ↓
Step 5: 强制引导 - 拖拽卡牌上阵（高亮前排位置）
         ↓
Step 6: 强制引导 - 开始战斗（点击"战斗"按钮）
         ↓
Step 7: 自动战斗演示（展示伤害数字、技能特效）
         ↓
Step 8: 强制引导 - 点击"主动技能"按钮释放技能
         ↓
Step 9: 战斗胜利 → 展示奖励（金币、经验、碎片）
         ↓
Step 10: 解锁"主城"界面 → 进入主城
```

### 3.2 教学关详细设计

#### 1-1 关卡（教学关）

| 参数 | 值 | 说明 |
|---|---|---|
| 关卡类型 | 教学关 | 不消耗体力 |
| 敌方阵容 | 2 只冰原狼 | HP 极低，保证玩家胜利 |
| 我方阵容 | 系统预设 3 张卡 | 不能自定义 |
| 特殊规则 | 无 | 纯教学 |

#### 教学步骤触发器

| 步骤 | 触发条件 | 引导方式 | 说明 |
|---|---|---|---|
| 1 | 进入阵容界面 | 黑色遮罩 + 高亮前排位置 | 引导拖拽卡牌 |
| 2 | 卡牌上阵完成 | 遮罩消失 + 高亮"战斗"按钮 | 引导开始战斗 |
| 3 | 进入战斗 | 黑色遮罩 + 高亮敌方目标 | 引导观察战斗 |
| 4 | 第 1 回合结束 | 遮罩 + 高亮"主动技能"按钮 | 引导释放技能 |
| 5 | 战斗胜利 | 全屏奖励展示 | 奖励反馈 |

### 3.3 教学关后引导

```
战斗胜利 → 返回主城 → 弹出"抽卡教学"引导
    ↓
引导点击"抽卡"入口
    ↓
赠送 10 抽免费券 → 引导单抽
    ↓
必定获得 1 张传说卡（新手池）
    ↓
引导查看新卡牌详情
    ↓
引导将新卡牌上阵替换弱卡
    ↓
教学完成，玩家可自由探索
```

## 四、阶段二：养成系统教学（Day 1-3）

### 4.1 Day 1 后续引导

| 时间点 | 引导内容 | 触发条件 |
|---|---|---|
| 首次战斗胜利后 | 引导查看卡牌详情 | 自动弹窗 |
| 首次抽卡后 | 引导升星系统 | 自动弹窗 |
| 首次升星后 | 引导装备系统 | 自动弹窗 |
| 首次装备强化后 | 引导技能升级 | 自动弹窗 |
| 完成日常任务 3 个 | 引导任务系统 | 红点提示 |
| 玩家等级 5 | 解锁爬塔入口 | 弹窗通知 |
| 玩家等级 10 | 解锁 PVP 入口 | 弹窗通知 |

### 4.2 养成引导步骤

#### 升星引导

```
Step 1: 引导点击"阵容"入口
Step 2: 引导点击某张卡牌
Step 3: 引导点击"升星"按钮
Step 4: 展示升星消耗（碎片 + 金币）
Step 5: 引导确认升星
Step 6: 播放升星动画（光效 + 星级提升）
Step 7: 展示属性提升对比
```

#### 装备引导

```
Step 1: 引导点击"装备"标签
Step 2: 引导点击空装备槽
Step 3: 展示可用装备列表
Step 4: 引导选择一件装备
Step 5: 引导点击"强化"按钮
Step 6: 播放强化动画
Step 7: 展示属性提升
```

## 八、引导关闭与跳过机制

### 8.1 可跳过条件

| 引导类型 | 可跳过条件 | 说明 |
|---|---|---|
| 教学关引导 | 不可跳过 | 所有新玩家必须完成 |
| 养成引导 | 完成教学关后可跳过 | 老玩家/回归玩家可跳过 |
| 功能解锁引导 | 有"知道了"按钮 | 一次性提示 |
| 每日签到引导 | 可关闭 | 关闭后不再提醒 |

### 8.2 引导状态存储

```typescript
interface TutorialState {
  // 教学关进度
  tutorial_1_1_completed: boolean;  // 1-1 关卡
  tutorial_1_2_completed: boolean;  // 1-2 关卡
  tutorial_gacha_completed: boolean;  // 抽卡教学
  
  // 养成引导
  tutorial_star_up_completed: boolean;
  tutorial_equip_completed: boolean;
  tutorial_skill_completed: boolean;
  
  // 社交引导
  tutorial_guild_completed: boolean;
  tutorial_friend_completed: boolean;
  tutorial_arena_completed: boolean;
  
  // 标记（防止重复弹窗）
  shown_tips: string[];  // 已显示的提示 ID
}
```
