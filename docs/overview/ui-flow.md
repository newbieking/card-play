### 2.1 启动/登录界面

```
┌─────────────────────────────┐
│                             │
│     [游戏 Logo]             │
│                             │
│     [背景动画]              │
│                             │
│  ┌─────────────────────┐   │
│  │    游客登录          │   │
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │    微信登录          │   │
│  └─────────────────────┘   │
│  ┌─────────────────────┐   │
│  │    Apple 登录        │   │
│  └─────────────────────┘   │
│                             │
│  版本号: v1.0.0             │
│  服务协议 | 隐私政策        │
└─────────────────────────────┘
```

**流程：**
1. 启动 → 检查本地 Token
2. Token 有效 → 直接进入主城
3. Token 无效 → 显示登录界面
4. 游客登录 → `POST /api/v1/account/guest_login`
5. 登录成功 → 获取档案 + 配置版本 → 进入主城

### 2.5 战斗界面

> **战斗表现方案：卡牌 + 技能特效（已决策）**
> - 战斗界面展示精美卡牌（有呼吸动画），无 2D 人物精灵图
> - 攻击方式根据职业区分：近战职业（战士/射手）卡牌飞向目标撞击，远程职业（法师/术士/辅助）原地释放技能特效

### 2.6 战斗动画系统

#### 2.6.1 卡牌在战斗中的状态

```
┌─────────────────────────────────────────┐
│           卡牌战斗状态机                  │
├─────────────────────────────────────────┤
│                                         │
│  [待机] ──行动──→ [攻击中] ──返回──→ [待机] │
│    ↑                  │                  │
│    │                  ↓                  │
│    └──────── [受击] ←─┘                  │
│                                         │
│  特殊状态：                              │
│  [冰冻] → 卡牌变蓝+冰封特效，跳过行动     │
│  [眩晕] → 卡牌旋转星星，跳过行动          │
│  [死亡] → 卡牌碎裂消散                    │
└─────────────────────────────────────────┘
```

#### 2.6.2 攻击动画分类

| 职业类型 | 攻击方式 | 动画表现 | 特效资源 |
|---|---|---|---|
| **战士** | 近战撞击 | 卡牌飞向目标 → 撞击 → 冲击波 → 返回原位 | 撞击特效（通用） |
| **射手** | 远程射击 | 卡牌轻微前倾 → 射出箭矢/子弹 → 命中目标 | 射击特效（通用） |
| **法师** | 原地施法 | 卡牌发光 → 手中法杖/法球亮起 → 全屏技能特效 | 元素特效（冰/火/雷等） |
| **术士** | 原地施法 | 卡牌暗影涌动 → 释放暗影/毒雾 → 目标出现debuff特效 | 暗影/毒雾特效 |
| **辅助** | 原地施法 | 卡牌圣光闪烁 → 光柱/治愈波 → 目标出现增益特效 | 圣光/治愈特效 |

#### 2.6.3 技能特效体系（按元素分类）

```
特效资源可复用：同一元素共用一套特效，不同技能只调整参数（大小/颜色/速度）

火系特效（可复用）：
├── 火球术（单体）    → 小型火球飞行 + 爆炸
├── 火焰风暴（AOE）   → 大型火焰漩涡 + 全屏震动
├── 灼烧（DOT）       → 目标持续燃烧
└── 护盾特效（通用）  → 卡牌周围火焰护盾

冰系特效（可复用）：
├── 冰锥术（单体）    → 冰锥飞行 + 碎裂
├── 暴风雪（AOE）     → 大型冰晶风暴
├── 冰冻（控制）      → 目标被冰封
└── 冰霜护盾（通用）  → 卡牌周围冰晶护盾

雷系特效（可复用）：
├── 雷击（单体）      → 闪电劈下
├── 雷霆万钧（AOE）   → 多道闪电
└── 减速（控制）      → 目标被电弧缠绕

暗系特效（可复用）：
├── 暗影箭（单体）    → 暗影能量飞行
├── 暗影爆发（AOE）   → 暗影能量扩散
└── 沉默（控制）      → 目标被暗影笼罩

光系特效（可复用）：
├── 圣光术（单体治疗） → 光柱从天而降
├── 群体治疗（AOE）   → 光环扩散
└── 复活（特殊）      → 目标从地面升起

通用特效：
├── 撞击（近战）      → 冲击波 + 碎片
├── 护盾吸收          → 护盾碎裂
├── 暴击              → 屏幕震动 + 放大数字
└── 击杀              → 目标碎裂消散
```

#### 2.6.4 特效资源清单

| 特效类型 | 资源格式 | 数量（预估） | 说明 |
|---|---|---|---|
| 元素技能特效 | PNG 序列帧 / WebM | 6 元素 × 3 = 18 套 | 单体/AOE/控制各一套 |
| 通用特效 | PNG 序列帧 / WebM | 10 套 | 撞击/暴击/护盾/死亡等 |
| Buff/Debuff 图标 | PNG | 20 个 | 状态效果图标 |
| 伤害数字 | PNG 序列帧 | 3 套 | 普通/暴击/治疗 |
| 屏幕震动 | 代码控制 | — | 暴击/大招时触发 |

#### 2.6.5 动画时间轴（单次攻击）

```
t=0ms    攻击方卡牌开始移动/发光
t=200ms  技能特效开始播放
t=400ms  伤害数字弹出
t=500ms  受击方闪烁/受击动画
t=600ms  受击方血条更新
t=800ms  攻击方返回原位（近战）/ 待机（远程）
t=1000ms 下一个行动单位开始

AOE 技能时间轴：
t=0ms    攻击方卡牌发光
t=200ms  全屏特效开始
t=500ms  所有目标同时弹出伤害数字
t=600ms  所有目标同时闪烁
t=800ms  特效消散
t=1000ms 下一个行动单位开始
```

## 三、界面流转规则

### 3.1 返回逻辑

| 当前界面 | 返回目标 | 说明 |
|---|---|---|
| 任何二级界面 | 上一级 | 标准返回 |
| 主城 | 退出确认弹窗 | 防误触退出 |
| 战斗中 | 暂停菜单（不能直接退出） | 战斗中退出判负 |
| 结算界面 | 主城 | 固定返回主城 |
