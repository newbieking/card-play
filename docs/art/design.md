卡牌收集、合成养成型游戏的核心要素
这类游戏的核心循环是：收集 → 合成/升级 → 养成 → 竞技/展示。以下是完整的分析：

一、核心玩法循环

获取新卡牌 → 强化/合成 → 数值提升 → 挑战/对战 → 获得奖励 → 重复
二、必备素材分类
1. 角色/生物类素材
类型	说明	示例
基础卡牌插画	角色正面全身/半身像	冰霜女王、暗影术士
角色表情集	不同情绪状态	战斗怒吼、施法专注、胜利微笑
Q版头像	UI头像、小图标用	圆形/方形小头像
待机动画帧	呼吸、眨眼、微动作	用于卡牌展示页
死亡/受伤帧	战斗反馈视觉	碎裂、发光消散
2. 特效/VFX 类素材
类型	说明	示例
入场特效	抽卡/上场时的视觉冲击	金色冲击波 + 全屏震动
攻击特效	普攻/技能的命中反馈	冰晶碎裂、邪能爆炸
受击特效	被攻击时的闪烁/粒子	红色火花、屏幕震动
升级特效	强化/进化时的视觉变化	光环扩散、粒子螺旋上升
边框呼吸光	稀有度的动态标识	传说牌金色脉冲
胜利/失败特效	结算画面	金币雨、卡牌碎裂
3. 场景/背景类素材
类型	说明	示例
对战棋盘	双方出牌区域	炉石式格子棋盘
职业专属场景	不同职业的战场氛围	冰原、暗影裂隙、战场废墟
卡背设计	玩家个性化展示	职业主题纹样
背景粒子	氛围填充	飘雪、灰烬、光点
4. UI/UX 类素材
类型	说明	示例
卡牌边框	按稀有度区分	普通灰/稀有蓝/史诗紫/传说金
按钮/面板	通用交互元素	开始、设置、商店
数值标识	攻击力/生命值显示	红色火焰图标、蓝色盾牌图标
资源图标	金币、钻石、经验药水	小型圆形图标
等级/星级标识	角色成长进度	星星、皇冠、徽章
5. 合成/进化类素材
类型	说明	示例
碎片图标	分解获得的材料	碎纸片、能量块
合成光效	两张卡融合的视觉	两道光束交汇、爆炸闪光
进化前后对比	形态变化参考	幼体→成体、普通→觉醒
觉醒/突破特效	突破等级上限时	全屏幕光环、天空裂开
三、按稀有度分级制作
稀有度	素材复杂度	特效层级
普通 (Common)	静态插画，无动画	无特效
稀有 (Rare)	增加细节纹理，待机微动	蓝色粒子飘散
史诗 (Epic)	多表情版本，进阶动画	紫色光环 + 粒子螺旋
传说 (Legendary)	全表情集 + 完整动画帧	全屏特效 + 角色动画 + 音效

五、技术格式建议
素材类型	推荐格式	备注
卡牌插画	PNG 透明底	方便叠加 UI
特效	PNG 序列帧 / WebM	透明通道保留
图标/UI	SVG / PNG	矢量优先
卡背	PNG	可加轻微纹理
总结：卡牌收集类游戏的素材核心围绕 "视觉差异化" —— 每张卡都要通过画风、特效、稀有度标识来制造收集欲和成就感。特效不仅是视觉装饰，更是数值成长的反馈信号。
