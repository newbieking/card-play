## 一、数值设计原则

### 1.1 属性分配原则

| 职业 | 主属性 | 副属性 | 定位 |
|---|---|---|---|
| 战士 | ATK + DEF | HP | 前排坦克，物理输出 |
| 法师 | MATK | SPD | 后排 AOE，法术输出 |
| 术士 | MATK | HP | 后排控制/削益 |
| 射手 | ATK + SPD | CRIT% | 后排持续输出 |
| 辅助 | MATK | HP/DEF | 后排治疗/增益 |

### 1.2 稀有度属性倍率

| 稀有度 | 基础属性倍率 | 最大星级 | 定位 |
|---|---|---|---|
| 普通 (Common) | ×0.6 | 5★ | 前期过渡 |
| 稀有 (Rare) | ×0.8 | 5★ | 中期主力 |
| 史诗 (Epic) | ×1.0 | 7★ | 后期核心 |
| 传说 (Legendary) | ×1.3 | 10★ | 终极追求 |

### 1.3 属性计算公式

```
最终属性 = base × stat_growth(level) × star_multiplier(star) × (1 + quality_variant)

其中：
  stat_growth(level) = 1 + 0.02 × (level - 1)   // 每级 +2%
  star_multiplier(star) = 1 + 0.175 × (star - 1)  // 每星 +17.5%
  quality_variant ∈ [-0.02, +0.02]                // 个体值浮动 ±2%
```

## 三、面板属性验证

### 3.1 冰霜女王（传说·法师）满配面板计算

```
基础值（1★1级）：HP=800, ATK=30, MATK=320, DEF=100, MDEF=160, SPD=60

假设：Lv.50, 7★, quality_variant=+0.015

stat_growth(50) = 1 + 0.02 × (50-1) = 1.98
star_multiplier(7) = 1 + 0.175 × (7-1) = 2.05

HP    = 800  × 1.98 × 2.05 × 1.015 = 3285
ATK   = 30   × 1.98 × 2.05 × 1.015 = 123
MATK  = 320  × 1.98 × 2.05 × 1.015 = 1314
DEF   = 100  × 1.98 × 2.05 × 1.015 = 411
MDEF  = 160  × 1.98 × 2.05 × 1.015 = 657
SPD   = 60   × 2.05 = 123

战力 = (HP×0.5 + ATK + MATK + DEF×0.8 + MDEF×0.8) × (1 + SPD×0.01)
     = (1642 + 123 + 1314 + 329 + 526) × 2.23
     = 3934 × 2.23 = 8773
```

### 3.2 铁卫守卫（稀有·战士）满配面板计算

```
基础值（1★1级）：HP=1200, ATK=180, MATK=20, DEF=200, MDEF=80, SPD=45

假设：Lv.50, 5★, quality_variant=-0.005

stat_growth(50) = 1.98
star_multiplier(5) = 1 + 0.175 × 4 = 1.70

HP    = 1200 × 1.98 × 1.70 × 0.995 = 4003
ATK   = 180  × 1.98 × 1.70 × 0.995 = 601
MATK  = 20   × 1.98 × 1.70 × 0.995 = 67
DEF   = 200  × 1.98 × 1.70 × 0.995 = 668
MDEF  = 80   × 1.98 × 1.70 × 0.995 = 267
SPD   = 45   × 1.70 = 76.5

战力 = (2001 + 601 + 67 + 534 + 214) × 1.765
     = 3417 × 1.765 = 6031
```

## 四、伤害模拟验证

### 4.1 冰霜女王 vs 铁卫守卫（冰克自然，+25%）

```
攻击方：冰霜女王 MATK=1314，冰元素
防御方：铁卫守卫 DEF=608，MDEF=267，自然元素
技能：冰锥术（法术伤害，倍率 1.5）

元素克制：冰 → 自然？ 不在克制循环中（火→冰→雷→水→火），自然中立
实际克制倍率：0%（自然不被冰克制）

基础伤害 = MATK × 倍率 = 1314 × 1.5 = 1971
减伤率 = MDEF / (MDEF + 500) = 267 / 767 = 34.8%
最终伤害 = 1971 × (1 - 0.348) = 1285
浮动后 ≈ 1221 ~ 1349
```

### 4.2 火焰先知 vs 冰霜骑士（火克冰，+25%）

```
攻击方：火焰先知 MATK=290，火元素
防御方：冰霜骑士 DEF=180, MDEF=100，冰元素
技能：火焰风暴（法术伤害，倍率 2.0）

元素克制：火 → 冰，+25%

基础伤害 = 290 × 2.0 = 580
元素加成 = 580 × 1.25 = 725
减伤率 = 100 / 600 = 16.7%
最终伤害 = 725 × (1 - 0.167) = 604
```

### 4.3 雷霆游侠 vs 暗影编织者（雷电 vs 暗，无克制）

```
攻击方：雷霆游侠 ATK=280，雷电元素，CRIT%=18%，CDMG%=1.5
防御方：暗影编织者 DEF=110，暗元素
技能：雷霆一击（物理伤害，倍率 1.8）

元素克制：雷电 vs 暗，无克制关系，0%

基础伤害 = 280 × 1.8 = 504
减伤率 = 110 / 610 = 18.0%
非暴击伤害 = 504 × (1 - 0.18) = 413
暴击伤害 = 413 × 1.5 = 620
期望伤害 = 413 × 0.82 + 620 × 0.18 = 339 + 112 = 451
```

