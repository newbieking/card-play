# Phase 0: 核心卡牌插画提示词

> 本文件夹存放10张核心卡牌的AI图像生成提示词
> 按照 `prompt-design-plan.md` 执行

## 文件命名规范
- `card_{card_id}_prompt.md` - 卡牌提示词文件
- `card_{card_id}_variants/` - 变体文件夹（存放3-5个变体）

## 执行顺序
1. **冰霜女王 (frost_queen)** - 传说·法师·冰 - 定风格用
2. 铁卫守卫 (iron_guardian) - 稀有·战士·自然
3. 冰霜骑士 (frost_knight) - 史诗·战士·冰
4. 火焰先知 (fire_oracle) - 史诗·法师·火
5. 暗影编织者 (shadow_weaver) - 史诗·术士·暗
6. 毒雾女巫 (poison_witch) - 稀有·术士·自然
7. 雷霆游侠 (thunder_ranger) - 史诗·射手·雷电
8. 风行斥候 (wind_scout) - 稀有·射手·自然
9. 圣光先知 (light_oracle) - 传说·辅助·光
10. 自然贤者 (nature_sage) - 稀有·辅助·自然

## 风格锚定
- 参考风格：《原神》角色立绘 / 《崩坏3》卡牌风格
- 视觉特征：半写实比例（7-8头身）、高饱和度、高对比度光影、东方奇幻元素
- 输出规格：原图 1536×2048 (3:4)，游戏内 768×1024 (3:4)
